### Get Started
1. Clone the repository:  
   ```bash
   git clone https://github.com/Dredarty/stock-portfolio-tracker.git
   ```

2. Install dependencies:  
   ```bash
   npm install
   ```

3. Start the application:  
   ```bash
   npm start
   ```

4. Access the application locally at: `http://localhost:3000`
